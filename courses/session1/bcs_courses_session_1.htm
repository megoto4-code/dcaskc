Course No - BCS 101

Group – A: Digital Electronics & Fundamental of Computers

Unit – I

Basic Electronics: Diode as a Rectifier, Transistor characteristics, Amplifiers, Multi-vibrators, Basics of IC Technology.

Unit – II

Number System, Boolean algebra and Logic Gates, Simplification of Boolean Functions – Map Method, Two and Three variable Maps, Product of Sums, Simplification, NAND and NOR implementation of Logic Gates, Don’t Care Condition, The Tabulation Method, Determination of Prime-implicants.

Unit III

Combinational Logic: Introduction, Design Procedures, Adders, Subtractors, Code Conversion, Analysis Procedure, Multilevel NAND Circuits, Exclusive-OR and Equivalence Functions, Binary Parallel Adder, Decimal Adder, Magnitude Comparator, Decoder, Multiplexers, Read Only Memory (ROM), Programmable Logic Array (PLA).

Unit IV

Sequential Logic Circuit: Introduction, Flip-Flops, Analysis of Clocked Sequential Circuits, State Reduction and Assignment, Flip-Flop Excitation table, Design Procedure, Design of Counters, Design with State Equations.

Unit V

Registers, Counters, Memory Unit: Introduction, Registers, Shift Registers, Ripple Counters, Synchronous Counters, Timing Sequences, The Memory Unit, Examples of Random Access Memory.

Text Books:-

  1. M. Mano, Digital Design, PHI
  2. Malvino A.P, Digital Principles and Applications, Tata McGraw Hill

Reference Books:-

  1. PVS Rao, Perspectives in Computer Architecture, PHI.
  2. Malvino & Leach, Digital Principles and Applications, Tata McGraw Hill Edition.
  3. R.P. Jain, Modern Digital Electronics, Tata McGraw Hill.
  4. T.C. Bartee, Digital Computer Fundamental, McGraw Hill.

Course No - BCS 101

Group B: Computer System Architecture 

Unit I

Register Transfer and Micro-operation – Register Transfer Language, Bus & Memory Transfer; Arithmetic, Logic & Shift Micro-operation, Arithmetic Logic Unit, Computer organization design – Computer Instruction with details like timing, control and Instruction Cycle, Memory Reference; Input and Output and Interrupt Instructions, Design of a Basic Computer.

Unit II

Programming the Basic Computer – Machine and Assembly Language with Programming Details. Micro-programmed Control – Control Memory, Address Sequencing, Micro-program Examples, Design of Control Unit.

Unit III

Central Processing Unit – General Register Organization, Stack Organization, Instruction Format, Addressing Modes, Data Transfer and Manipulation, Program Control, Length and type of instruction, RISC and CISC.

Unit IV

Computer Arithmetic – Addition, Subtraction, Multiplication, Division Algorithms with Hardware implementation, Floating point and Decimal Arithmetic operation.

Unit V

Input output Organization – Input Output Interface, Asynchronous Data Transfer, Modes of Transfer, Priority Interrupt, DMA, Input Out Processor, Serial Communication.

Advances in Memory System – Memory hierarchy, Different Memory Organization (Main, Auxiliary, Associative, Cache) and Virtual Memory.

Text Books:-

  1. M. Mano, Computer System Architecture, Pearson Education Asia/PHI.
  2. V. Carl Hamacher, Zvonko G. Vranesic, Safwat G. Zaky, Computer Organization, McGraw Hill.

Reference Book:-

1.      PVS Rao, Perspectives in Computer Architecture, PHI.

Course No - BCS 102

Group – A: Data Structures

Unit I

Data type, Abstract Data type, Data Structure, Storage Structure, Complexity of an Algorithm, Big ‘O’ Notation; Arrays, Linear Lists – Representation, operations and implementation of Singly and Doubly Linked list.

Unit II

Stack – Representation, Operations: Array and Linked List implementation; Queue – Representation, operations: array and linked list implementation of Single, Multiple, Priority Queue, Deque and Circular Queue; Application of Stacks and Queues, Recursion.

Unit III

Trees – Definitions and Concepts; Storage Representation and Manipulation of General Trees, Binary Trees, Threaded Binary Trees, AVL Trees, B – Trees, Red Black Trees, Multilinked Structures; Applications.

Unit IV

Graphs: Representation, Traversal and other operations.

Tables: Searching, Sequential Table, Hash Tables and Symbol Tables.

Files: Properties of Physical Storage Media, File organization Techniques.

Unit V

Sorting – Selection, Insertion, Bubble, Shell, Quick, Heap, Merge, Radix and Address Calculation sorts; Searching – Sequential, Binary and Interpolation search, Comparison trees.

Text Books:-

1.      J.P. Trembly and P.G. Soreson, An Introduction to Data Structures with Applications, Tata McGraw Hill

2.      Robert L. Kruse, Data Structures and Program Design, PHI Pvt. Ltd.

3.      E. Horowitz and S. Sahni, Fundamentals of Data Structures, Galgotia Book Source.

Reference Books:-

1.      Tanenbaum, Y. Langsam, A.J. Augenstein, Data Structure using C, PHI, 1990.

2.      E. Horowitz and S Sahni, Data Structures with PASCAL 3rd Edition, Galgotia, 1991.

3.      A. Aho and J.E. Hocraft and J.D. Ullman, Data Structure and Algorithms, Addison Wesley.

4.      Yashwant Kanitkar, Data Structure using C, BPB Publication.

 

Course No - BCS 102

Group – B: Programming with FORTRAN & C

Unit I

Procedure Oriented Programming (Independent of Languages) – Notion of an Algorithm, Tools for design and analysis of Algorithms – Flowchart, Decision Table, Pseudo-code.

Programming in FORTRAN I: Constant and Variables, Arithmetic Expressions, Input Output Statement, Control Statements, The DO statement, Subscripted Variables, Logical Expressions and Operators.

Unit II

Programming in FORTRAN II: Functions and Subroutines, Character Manipulation, File Processing, Miscellaneous features of FORTRAN 77, Introduction to FORTRAN 90/95 

Unit III

Programming in C I: Identifiers, variables; Declarations, operators and Expressions, Data Input and Output; Control Statements – Branching, Looping, Nested control structures; Recursion.

Unit IV

Programming in C II: Functions, Structures and Unions, Pointers, Passing Arrays; Pointers; Structures and Function to other function.

Unit V

Programming in C III: File Processing, Enumeration, Command Line Parameters, Macros, The C Preprocessor.

Text Books:-

  1. E. Balaguruswamy, Programming in ANSI C, Tata McGraw Hill.
  2. Byron S. Gottfried, Programming with C, Tata McGraw Hill Edition.
  3. Ram Kumar, Programming with FORTRAN 77, Tata McGraw Hill.
  4. Rajaraman, Computer Programming in FORTRAN 90 and 95, PHI.

Reference Books:-

  1. Yashwant Kanitkar, Let us C, BPB Publication.

  

BCS 103: PRACTICAL ON DATA STRUCTURE AND PROGRAMMING WITH FORTRAN AND C.

Minimum 10 (ten) experiments are to be performed based on the Theory paper.

 

Course No - BCS 201

Group – A: Microprocessors and Assembly Language Programming

Unit I

Evolution of Microprocessor, Architecture of Intel 8085, Addressing Modes, Instruction Sets and Instruction formats.

Unit II

Intel 8086 – Architecture. 8086 Addressing Modes, 8086 Instruction Sets, 8086 Assembler Dependent Instruction, 8086 I/O, System design using 8086 and typical 8086 Assembler Pseudo instructions.

Unit III

Programming Model: General Purpose Registers, Pointer and Index Registers; Flags; Segment Registers; Program invisible registers; Memory Addressing and Addressing models.

Unit IV

Memory Interfacing: Memory Address Decoding; Memory mapped and I/O mapped I/O, Interrupt, Vectored Interrupt, Study of 8259 programmable interrupt controller and its interfacing to 8086. Basic I/O Interface: Basic I/O and handshaking, I/O port address decoding.

Unit V

8255 programmable peripheral interface; 8279 programmable keyboard and display interface; Direct Memory Access, 8237/8257 DMA controller, Video Controller. Serial and Parallel data communication and interface standards. 

Text Books - 

  1. Ramesh S. Gaonkar, Microprocessor Architecture, Programming and Applications with the 8085, Penram International.
  2. A. P. Mathur, Introduction to Microprocessors, Tata McGraw Hill.
  3. Rafiquzzaman, Microprocessor – Theory and Applications, PHI.
  4. Ajit Paul, Microprocessors – Principles and Applications, Tata McGraw Hill.

 

Course No - BCS 201

Group – B: Object Oriented Programming (C++ & JAVA)

Unit I

Object Oriented Programming concepts (Independent of languages) Classes and Methods, Message, Instances and Initialization, Inheritance and types, Static and Dynamic Binding, Replacement and Refinement, Polymorphism, Visibility and Dependency, Encapsulation.

Unit II

Object oriented programming in C++: An overview of C++, Class and Objects, Arrays, Pointers and Reference, Function and Operator Overloading, Inheritance.

Unit III

Object oriented programming in C++: Virtual functions and Polymorphism C++ I/O system basics, C++ file I/O, Array Based I/O, Templates, Exception handling.

Unit IV

Object oriented programming in JAVA: Genesis of JAVA; An overview of JAVA, Data Types, Variables and Arrays, operators, control statements, introducing classes, and methods.

Unit V

Object oriented programming in JAVA: Inheritance, packages and interfaces, Exception handling, Multithreaded Programming, I/O, Applets.

Text Books –

  1. Timoth Budd, An Introduction to Object Oriented Programming, Addison Wesley Publishing Company (for unit I).
  2. Herbert Schildt, C++, The Complete Reference, Tata McGraw Hill Publishing Company (for Unit II & III).
  3. Patrick Naughton and Herbert Schildt, JAVA, The Complete Reference, Tata McGraw Hill (for Unit IV & V).
  4. E. Balaguruswamy, Object Oriented Programming with C++, Tata McGraw Hill.

Reference Books –

1.      Samanta, Object Oriented Programming with C++ and JAVA, PHI Publication.

 

Course No - BCS 202

Group – A: Discrete Mathematics

Unit I

Mathematical Logic: Statements and Notations, Connectives, Normal forms. The Theory of Inference for the Statement Calculus, The Predicate Calculus, Inference theory of the Predicate Calculus.

Unit II

Set Theory – Basic concept of Set Theory, Representation of Discrete Structures, Relations and ordering, Functions, Natural Numbers, Recursion, Recursion in Mechanical Theorem proving.

Unit III

Algebraic Structures: Algebraic systems: Examples and General properties, Semi-groups and Monoids, Grammars and Languages, Polish Expressions and their Compilations, Groups, The application of Residue Arithmetic to Computers Group Codes.

Unit IV

Lattices and Boolean Algebra with applications: Lattices and Partially ordered sets, Boolean Algebra, Boolean functions, Representation and Minimization of Boolean functions, Design Examples using Boolean Algebra, Finite state Machines.

Unit V

Graph Theory with application: Basic concepts of Graph Theory, Storage Representation and Manipulation of Graphs, Simple Precedence Grammars, Fault Detection in Combinational Switching Circuit, PERT and CPM Techniques.

Text Books –

1.      J.P. Trembly and R. Manohar, Discrete Mathematical structures with Applications to Computer Science, Tata McGraw Hill International Edition 1987.

2.      Narsing Deo, Graph Theory and its Application to Engineering and Computer Science, PHI.

Reference Books –

1.      Goodaire and Parmenter, Discrete Mathematics with Graph Theory, EEE.

 

Course No - BCS 202

Group – B: Database Management System

 Unit I

Records and Files, Advantages of Database Management System (DBMS), Architecture for Database System, Components of Database Management System (DBMS), Data Models: Relational, Hierarchical and Network, An introduction to Relational Database, Relational Data Integrity, E-R diagram.

Unit II

Query Languages: Relational Algebra, Relational Operators, Relational Calculus, The SQL language, Views. 

Unit III

Functional Dependencies and Normal forms: 1NF, 2NF, 3NF, BCNF, Higher Normal Forms.

Unit IV

Indexing, Hashing, Query Processing and Optimization.

Unit V

Transaction Concept, Recovery, Concurrency, Security.

Text Books –

1.      Silberschatz, Korth and Sudarshan, Database System Concept, McGraw Hill.

2.      C.J. Date, An Introduction to Database System, Addison Wesley Publishing Company.

3.      R. Elmasri and S.B. Navathe, Fundamental of Database System, Addison Wesley.

Reference Books –

  1. J.D Ullman, Principles of Database Systems, Galgotia Publications.
  2. Hansen and Hansen, Database Management System, PHI.

 

Course No - BCS 203

Group – A: System Analysis and Design

Unit I

The System Concept, Elements of a system, types of system. Introduction to system Development Life Cycle, Recognition of need, prototyping, Roles of System Analyst, The analyst/ user interface.

Unit II

Introduction to system analysis, determining the user’s information requirements, problem definition, Background analysis, fact-finding, fact analysis. Introduction to structured analysis, the tools of structured analysis, Feasibility study; oral representation, Data analysis, Cost/Benefit analysis, the system proposal.

Unit III

Introduction to System Design; The process of design (logical and physical design), Design methodology, Structured design, structured walkthrough, Major development activities, Data Validation.

Introduction to input design, output design, form design.

File Structure, File organization, Database design and the role of DBA.

Unit IV

Introduction to system testing: The Test Plan, Quality assurance.

System Conversion, Post implementation review, Software maintenance.

Unit V

Procedure for Hardware/ Software selection, Project Management and Control, Project Control, Gantt Chart, PERT and CPM, System Security. 

Text Books –

  1. System Analysis and Design, Elias M. Awad, Galgotia Publication.

Reference Books –

  1. Kendall and Kendall, System Analysis and Design, PHI.
  2. Igor Hawryszkiewyez, Introduction to System Analysis and Design, PHI.

 

Course No - BCS 203

Group – B: Operating System Architecture

Unit I

Operating System overview: Operating System as an extended machine and resource manager, operating system classification; Operating system architecture: Operating system mode and system calls.

Processor Management Functions: Process model, Hierarchies and implementation; Process states and transitions, Multiprogramming, Multitasking, Multithreading, Levels of Schedulers and Scheduling algorithms; micro-kernel architecture.

Unit II

Memory Management functions: Memory management of single user operating system, Memory Management for multi-user operating system, partition, swapping, paging, segmentation, virtual memory.

Unit III

Device Management functions: I/O devices and controllers, interrupt handlers device, independent I/O software, user level I/O software; Disk Scheduling, clock hardware software; Terminal input output software.

Unit IV

File Management functions: naming structure, types, access mechanism, attributes and operations; Hierarchical directory system, Directory structure and directory operations; File space allocations; File sharing file locking; Symbolic links, files protection and security, Distributed file system.

Unit V

Concurrent Programming, sequential and concurrent process, precedence graph, Bernstein Conditions, time dependency, critical node section, mutual exclusion problem, classical coordination problem, Deadlock handling, inter-process communication.

Case Study: UNIX/ LINUX. 

Text Books –

  1. H.M. Deitel, An introduction to Operating System, Addison Wesley Publication.
  2. A.S. Tanenbaum, Modern Operating System, PHI.
  3. Peterson and Silberschatz, Operating System Concepts, Addison Wesley.
  4. Silberschatz and Galvin, Operating System Concepts, John Wiley and sons Pvt. Ltd.

Reference Books –

  1. Madnick and Donovan, Operating Systems, McGraw Hill.
  2. Ullman, Principles of Operating Systems, Galgotia Publication.

Course No - BCS 301

Group – A: Design and Analysis of Algorithms

Unit I

Introduction: Writing structured programs, Analyzing Algorithms, Stacks and Queues, Trees, Heaps and Heap sort, Graphs, Hashing.

Unit II

Divide and Conquer: The general concept, Binary search, Finding the maximum and minimum, Merge sort, Quick sort, Selection sort, Strassen’s matrix manipulation, Greedy methods, Minimum spanning trees.

Unit III

Dynamic programming: The general method, Multi stage graphs, All pairs shortest paths, Optimal binary search trees, Reliability design, The Traveling sales person problem, Flow shop scheduling.

Unit IV

Basic search and traversal techniques: The techniques, Code generation, AND/OR graphs, Game trees, Bi-connected components, Depth first search and Breadth first search.

Unit V

Backtracking: the Queens problems, Introduction to NP-Hard and NP-Complete problems.

Text Books –

  1. S. Sahni, Horowitz and Rajasekaran, Computer Algorithm, Galgotia Publications.
  2. Horowitz and Sahni, Fundamentals of Computer Algorithm, Galgotia Publications.
  3. Aho, Hoperoft and Ullman, Design and Analysis of Computer Algorithm, Addison Wesley.
  4. S.E. Goodman and S.T. Hedet, Introduction to the Design and Analysis of Algorithm.

Reference Books –

  1. S. K. Basu, Design Methods and Analysis of Algorithms, PHI.
  2. Gilles, Brassard and Paul Brantley, Algorithmic theory and practices, PHI.
  3. Knuth, Fundamental Algorithms, The art of Computer Programming, Vol-1
  4. S. Sahni, Analysis of Algorithms through C++, Tata McGraw Hill.

 

Course No - BCS 301

Group – B: Computer Networks and Data Communication

Unit I

Introduction, Topology, OSI models, Network models: Physical layer, Signals, Digital Transmission, Analog transmission, Multiplexing, transmission media, circuit switching, packet switching.

Unit II

Data link layer – Error detection and correction, Data link control and protocols, point to point access, multiple accesses, Local Area Network: Ethernet, connecting LANs, Back bone networks, virtual LANs, Satellite Networks.

Unit III

Network layer – Internet Networking, Addressing and routing, network layer protocols, unicast and multicast routing, gateways, switch, hub.

Unit IV

Transport layer – elements of transport protocols, Simple transport protocol, User Datagram protocol (UDR), Transmission control protocols (TCP), Congestion control and quality of service, performance issues 

Unit V

Application layer – Domain Name System (DNS), Electronic Mail, HTTP and WWW, Multimedia, Network Security 

Text Books

  1. Andrew S. Tanenbaum, Computer Networks, PHI Publications.
  2. Behrouz A. Forouzan, Data Communication and Networking, Tata McGraw Hill Editions.

Reference Books

  1. Michel A Gallo, Willan M Hancork, Computer Communications and Networking Technologies. Thomson Books/ cole.
  2. William Stallings, Data and Computer Communication, PHI Publications.
  3. Achyut S. Godbole, Data Communication and Networks, Tata McGraw Hill Edition.

 

Course No - BCS 302

Group – A: Numerical and Statistical Methods

Unit I

Numerical Methods – errors in numerical calculations – transcendental equation – introduction – Bisection Method – iteration method – method of false position – Newton – Raphson method.

Unit II

Interpolation – Newton’s formulae (forward and backward) for interpolation – Lagrange’s interpolation formula – simultaneous linear equations – Gauss Elimination and Gauss Jordan methods – Gauss Scidal method.

Unit III

Numerical integration – Trapezoidal and Simpson’s rule – differential equation – Euler, Runge-Kutta and predictor and corrector methods.. 

Unit IV

Mathematical Expectation – Variance – Covariance – Moment Generating Functions – theoretical distributions – Binomial, Poisson, Normal and Exponential Distributions – MGFS of these distribution – additive properties – Recurrence relations for the moment.

Unit V

Linear correlation and regression – properties of correlation and regression coefficients, numerical problems for finding the correlation and regression coefficients.

Text Books –

  1. S.S. Sastry, Introductory Methods of Numerical Ananlysis, PHI, New Delhi, 1982.
  2. M.K. Jain, S.R.K. Iyengar and R.K Jain, Numerical methods for Science and Engineering Computation, Wiley Eastern Limited – 2nd Editions – 1995.
  3. Gupta S.C. and Kapoor V.K. – Fundamentals of Statistics – Sultan Chand and Sons – New Delhi (1994).
  4. Bajpat A.C., Cal I.M. and Fairdy J.A., Statistical Mathods for Engineering and Scientists – John Wiley and Sons.

Reference Books –

  1. Rajaraman, Computer Oriented Numerical Methods, PHI.
  2. Venkataraman, Numerical Computations.
  3. Stoer, Bullrich, Computer Oriented Numerical Methods, Springer Verlag.
  4. Krishnamurthy, E.V. Sen, Computer based Numerical Algorithm, Easc West Press.

 

Course No - BCS 302

Group – B: Visual Basic

Unit I

Introduction to Visual Programming: Integrated Development Environment (IDE) – the Menu bar, the toolbar, the toolbox, the properties window, form designer, form layout. Creating, Saving, Editing, running the project, Elements of user interface, programming basics in visual basic, visual development and event driven programming. Customize the environment.

Unit II

VB programming fundamentals: variables, constants, arrays, collections, procedures, function return values, control flow statements, loop statements – for loop, do loop, while loop, nested control structures.

Working with forms appearance of forms, the startup forms, loading, showing and hiding forms, controlling one form within another. Design menus, programming menu commands, manipulating menus at runtime, building dynamic forms at runtime, drag and drop operation.

Unit III

Basic ActiveX Control: The Text Box control, list box and combo box control, searching a sorted list, the scroll bar and slider controls.

Drawing with Visual Basic: Graphics control, coordinate system, Graphics methods – drawing texts, drawing lines and shapes, filling shapes, manipulating colors and pixels with visual basics.

Unit IV

VB interfaces and an introduction of Advance Programming topics, Multiple Document Interface Applications, Interface Design – using visual design elements, working with files, using Object Linking and Embedding (OLE) to control other applications.

Unit V

Database Programming with Visual Basic: The Data Control, the data control properties, data control; methods, understanding relational concepts – the Primary Key, Foreign Key, using basic data managers, entering data, validation data and accessing fields in record sets: DAO, RDO and ADO, An introduction to SQL.

Text Books –

  1. Evagelos Petroutsos, Mastering Visual Basic, BPB
  2. Brain Siler and Jeff Spotts, Using Visual Basic 6, PHI, 2001
  3. VB Unleashed, Sam Publication
  4. Programming VB 6.0, The Complete Reference, TMH
  5. Stephen Holzner, VB Black Book
  6. Programming VB 6.0, Pearson Edition.

Course No - BCS 303

Group – A: Computer Graphics

Unit I

Development of Computer Graphics: Basic Graphics systems and standards; raster scan and random scan graphics; continual refresh and storage displays; display processor and character generator; colour display techniques, frame buffer and bit operations, concepts in raster graphics.

Unit II

Points, lines and curves; scan conversion: line drawing algorithms, circle and ellipse generations; polygon filling, conic section generation, antialiasing.

Unit III

Two dimensional viewing; basic transformation; coordinate systems, windowing and clipping; segments; interactive picture construction techniques; interactive input/output devices.

Unit IV

Three dimensional concept: 3D representation and transformation; 3D viewing, algorithm for 3-D volume, spine curve and surfaces, fractals; quadtree and octree structure; hidden line and surface rendering and animation.

Unit V

Computer Art - Introduction to Graphics packages (Photoshop, Flash), Animation, Morphing.

Text Books –

  1. Donal Hearn and M. Pauline Baker, Computer Graphics, PHI
  2. D.F Rogers, Procedural Elements of Graphics, McGraw Hill.

Reference Books –

  1. Steven Harrington, Computer Graphics – A Programming Approach, McGraw Hill International Editions.
  2. William M. Newman and Robert F. Sprowll, Principles of Interactive Computer Graphics, McGraw Hill International Editions.
